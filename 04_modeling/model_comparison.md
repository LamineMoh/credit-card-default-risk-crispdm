# Model Comparison

## Objective
Compare multiple supervised learning models to identify the best trade-off between performance and business risk.

## Models Evaluated
- Logistic Regression
- Decision Tree
- Random Forest
- Gradient Boosting

## Evaluation Criteria
- Recall (priority metric)
- F1-score
- Stability across validation sets

## Selection Rationale
The selected model balances high recall with acceptable precision and model robustness.
